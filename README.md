# Chat App (FastAPI + WebSockets)

Учебный проект (аналог мессенджера Telegram).  
Реализованы REST API и WebSocket для работы с пользователями, чатами и сообщениями.

---

## Стек
- Python 3.11+
- FastAPI
- SQLAlchemy (async)
- PostgreSQL
- Alembic (миграции)
- Docker, Docker Compose
- WebSockets
